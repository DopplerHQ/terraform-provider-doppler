# doppler_secret_note (Resource)

Manage the note attached to a Doppler secret. Notes are scoped per project and secret name, and they apply across all configs in the project.

The note resource is independent of `doppler_secret`. A note can be declared for a secret managed inside or outside Terraform, as long as the secret exists in at least one config in the project.

## Example Usage

```terraform
resource "doppler_secret_note" "stripe_key" {
  project = "backend"
  name    = "STRIPE_SECRET_KEY"
  note    = "Owner: payments team. Rotate quarterly via Stripe dashboard."
}
```
